Help

TeX Editor is a very basic (La)TeX editor app with a native conversor from a subset of (La)TeX to HTML.

If you like this app please considere support Mozilla MathML Project and the team that work on it.

Features

  • Text editor
  • HTML preview
  • Save file
  • Open file

(La)TeX Subset Support

  • Paragraphs
  • Inline math mode
  • Display math mode
  • Math TeX expressions
  • Math TeX environments
